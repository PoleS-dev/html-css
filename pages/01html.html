Cours sur les Formulaires HTML

Les formulaires en HTML permettent de recueillir des données auprès des utilisateurs. Ils sont définis à l'aide de la balise <form>.

Utilité des attributs et de la balise label

- La balise <label> est utilisée pour associer un texte descriptif à un champ de formulaire. Elle améliore l'accessibilité et l'expérience utilisateur.
- Les attributs des champs de formulaire permettent de spécifier des contraintes ou des valeurs par défaut. Ils aident à la validation des données saisies par l'utilisateur.

Attributs importants dans un formulaire

  • action : Définit l'URL où les données du formulaire seront envoyées.
  • method : Spécifie la méthode HTTP utilisée (GET ou POST).
  • name : Attribut utilisé pour identifier les champs du formulaire côté serveur.
  • required : Indique qu'un champ est obligatoire.
  • placeholder : Fournit un texte indicatif dans un champ de saisie.
  • min et max : Définissent les valeurs minimales et maximales pour les champs numériques.
  • maxlength et minlength : Spécifient la longueur minimale et maximale pour un champ texte.
  • value : Définit une valeur par défaut pour un champ.
