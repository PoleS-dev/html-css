Les formulaires HTML permettent aux users d'envoyer des données a un serveur.

1. Structure d'un formulaire

Un formumaire est defini par la balise form

Explication des attributs

name : Cet attribut est utilisé pour identifier un champ du formulaire. Quand le formulaire est soumis, les données sont envoyées sous form de paire clé-valeur où le name est la clé.

label for : l'attribut for du label est utilisé pour associer un libellé à un champs via son id

2. Type de champs

HTML offre plusieurs types de champs :

text-- Champs de texte

email-- Champs de email

password-- Champs de password

radio-- boutons radio

textarea-- zone de text

checkbox-- case à cocher

number-- entrée numérique

file -- upload de fichiers

submit -- bouton d'envoi

exo : faire son formualire -> champs nom, champs prenom, champs textarea, bouton submit et amusez vous à custom avec css

4. SEO et les formulaires HTML

Les formulaires jouent un rôle important en SEO, notamment pour l'experience utilisateur et l'accessibilité

label et for : améliorent l'accécibilité et la lisibilité pour les moteurs de rechrche

input=name : les champs bien nommés aident les outils d'analyse du site

l'attribut placeholder : fournit des indices visuels aux utilisateurs sans remplacer le label

Exemple de formulaire optimisé SEO :

    <form action="traitement.php" method="POST">
        <label for="nom">Nom:</label>
        <input type="text" id="nom" name="nom" placeholder="Votre nom" required>
        
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" placeholder="Votre email" required>
        
        <button type="submit">Envoyer</button>
    </form>
