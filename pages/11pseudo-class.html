Cours sur les Pseudo-classes en CSS

Les pseudo-classes en CSS permettent de styliser des éléments dans un état particulier, comme lorsqu'un utilisateur passe la souris dessus ou lorsqu'un champ de formulaire est actif.

Exemples de Pseudo-classes

  • :hover - S'applique lorsqu'un utilisateur survole un élément.
  • :focus - S'applique lorsqu'un élément, comme un champ de saisie, est activé.
  • :nth-child(n) - Cible un élément en fonction de sa position parmi ses frères.
  • :first-child - Sélectionne le premier enfant d'un élément parent.
  • :last-child - Sélectionne le dernier enfant d'un élément parent.
  • :checked - S'applique aux éléments cochés (cases à cocher, boutons radio).
  • :active - S'applique lorsqu'un élément est activé (ex: un bouton cliqué).

Exemples de Pseudo-éléments

  • ::before - Ajoute un contenu avant un élément.
  • ::after - Ajoute un contenu après un élément.
